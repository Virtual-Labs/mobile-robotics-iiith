Introduction

The virtual lab on Mobile Robotics is designed to supplement a senior undergraduate or graduate level course in Mobile Robotics. The experiments are designed to improve the clarity in understanding of the basic concepts as well as to enable the student to put them into practical use. The experiments constitute the quintessential algorithmic modules required for automating a robot, namely algorithms dealing with localization, mapping, planning, exploration and simultaneous localization and mapping.

The textbooks for this course are:

  1. Probabilistic Robotics by Sebastian Thrun, Wolfram Burgard and Dieter Fox
  2. Autonomous Robots by Illah Norbhaksh and Roland Siegwart
  3. Computational Principles of Mobile Robots by Michael Jenkin and GregoryDud

List of experiments
  • Sensor Modelling
  • Velocity Modelling
  • Mapping
  • Localization
  • Grid based navigation
  • Forward kinematics
  • Scan matching
  • RRT based path planning
  • Exploration
  • Monte Carlo localization
Target Audience

This virtual lab is suitable for senior undergraduate students and first-year graduate students taking a course in Mobile Robotics. As of other labs of this nature, this lab is designed to be a supplement for a course in Mobile Robotics.

Pre-requisites for this course

The student must know basic probability (Bayes Theorem) usually covered in an introductory probability course in most Indian syllabi.

Courses Aligned

  • Probabilistic Robotics
  • Robotics: Motion-Planning and Exploration

Prerequisite S/W

  • Java Virtual Machine (JVM)
